# Supabase SQL

## Punto de entrada

Para una base nueva usar solo [demo_bootstrap.sql](/Users/matiasvazquez/Documents/Glivac-demo/supabase/demo_bootstrap.sql:1).

Ese archivo crea:

- catálogo de roles demo en `app_roles`
- perfiles de usuario en `user_profiles`
- tablas operativas del sistema
- helpers de permisos y RLS
- stores iniciales para la demo
